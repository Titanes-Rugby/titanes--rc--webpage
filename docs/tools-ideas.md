# Tool Pages Ideas

Backlog de ideas para nuevas herramientas internas del sitio.

## 1) Fixture Creator
- Objetivo: crear partidos con equipo, fecha, sede y resultado.
- Output: bloque listo para web + copy para redes.
- Valor: acelera publicación de calendario y resultados.

## 2) News Builder
- Objetivo: crear noticias desde formulario (titulo, resumen, imagen, tags).
- Output: preview de card + JSON para integrar en `Media`.
- Valor: flujo editorial más rápido y consistente.

## 3) Sponsor Kit Generator
- Objetivo: gestionar patrocinadores por tier (`main`, `gold`, `support`).
- Output: sección completa (grid + marquee) con preview/export.
- Valor: mantenimiento comercial centralizado.

## 4) Team Roster Builder
- Objetivo: armar planteles por categoría desde form o CSV.
- Output: estructura de datos para `TeamsPage`.
- Valor: actualización de roster sin tocar código manualmente.

## 5) Social Post Generator (Recomendado)
- Objetivo: generar piezas para redes con templates.
- Formatos: `1:1`, `4:5`, `16:9`.
- Casos: match day, resultado final, MVP, convocatoria.
- Valor: impacto directo en difusión semanal.

## 6) Brand Theme Studio
- Objetivo: ajustar tokens visuales (colores, contrastes, tipografía).
- Output: preview de componentes UI en tiempo real.
- Valor: control de branding y consistencia visual.

## Prioridad sugerida
1. Social Post Generator
2. Fixture Creator
3. News Builder
4. Team Roster Builder
5. Sponsor Kit Generator
